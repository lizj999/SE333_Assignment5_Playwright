# SE333 Assignment 5 - Part 2: Playwright UI Testing

## Overview
This project contains automated UI tests for the DePaul University Bookstore
using Playwright and JUnit 5. Tests cover the full purchase pathway for the
JBL Quantum True Wireless Gaming Earbuds.
